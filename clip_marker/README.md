# ClipMarker

ゲームクリップからキル・デスなどのイベントを自動検出してマーカーを付けるツールです。

## 動作要件

- Windows 10 / 11
- Python 3.10 以上
- ffmpeg（動画チャプター埋め込み機能を使う場合のみ）

## セットアップ

```
pip install -r requirements.txt
```

ffmpeg が必要な場合は https://ffmpeg.org からダウンロードし、
`ffmpeg.exe` を PATH の通ったフォルダに置いてください。

## 起動

```
python app.py
```

または `ClipMarker.bat` をダブルクリック。

## 使い方

1. 「ファイルを開く」でゲームクリップ（MP4 / MOV / MKV 等）を選択
2. ゲームタイトルを選択（現在: Splatoon 2 / 3）
3. 検出設定を調整
   - **キル感度**: キルフィード変化の検出閾値（低いほど敏感）
   - **デス感度**: 画面暗転の検出閾値
   - **クールダウン**: 同一イベントの最小間隔（秒）
   - **サンプルFPS**: 解析する間引きFPS（高いほど精度↑・速度↓）
4. 「解析開始」をクリック
5. 右パネルで検出結果を確認・編集
   - Deleteキーでイベント削除
   - ダブルクリックでラベル編集
   - 「＋ 手動追加」で任意位置にマーカー追加
6. エクスポート
   - **CSV / JSON**: タイムスタンプ一覧
   - **動画+チャプター**: ffmpegでチャプターメタデータを埋め込んだMP4

## 検出の仕組み（Splatoon）

| イベント | 検出方法 |
|----------|----------|
| キル | 画面右上のキルフィード領域（ROI）のフレーム差分が閾値を超えたとき |
| デス | 画面中央部が急激に暗くなり、かつ輝度が40以下になったとき |

## カスタマイズ

`detector.py` の `SPLATOON_ROIS` を変更することで、  
異なる解像度・レイアウトに対応できます。  
他のゲームに対応させるには新しい Detector クラスを作り、  
`app.py` の `GAME_PROFILES` に登録してください。

## ファイル構成

```
clip_marker/
  app.py          メインGUI
  detector.py     フレーム解析・検出ロジック
  exporter.py     CSV / JSON / ffmpegエクスポート
  requirements.txt
  ClipMarker.bat  Windows用ランチャー
```
